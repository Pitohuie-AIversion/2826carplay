# WECHAT_REVIEW_AND_PRIVACY.md

## 1. 目的

本文件用于约束小程序在上线前需要考虑的审核、隐私和合规问题。

具体审核规则以微信公众平台当前要求为准。本文只作为项目内部开发约束，不替代正式审核要求。

## 2. 第一版隐私原则

第一版只收集预约所必需的信息：

- 姓名
- 手机号
- 取车日期
- 还车日期
- 城市
- 备注

禁止收集：

- 身份证照片
- 驾驶证照片
- 精确定位
- 通讯录
- 人脸信息
- 与预约无关的个人信息

## 3. 用户授权原则

需要用户主动触发的能力：

- 打电话
- 联系客服
- 分享
- 未来获取手机号
- 未来获取定位

禁止自动获取用户隐私信息。

## 4. 页面内容约束

页面内容应清晰说明：

- 车辆价格以客服确认为准
- 车辆档期以客服确认为准
- 提交预约不代表订单已确认
- 押金、保险、取还车规则需客服确认

## 5. 上线前检查

上线前需要检查：

- 小程序名称是否与页面品牌一致
- 服务类目是否与业务匹配
- 隐私政策是否覆盖实际收集字段
- 客服联系方式是否有效
- 电话号码是否有效
- 页面无诱导分享
- 页面无虚假折扣
- 页面无夸大承诺
- 图片素材拥有使用权限

## 6. 第一版禁止上线的功能

未完成正式合规准备前，不上线：

- 在线支付
- 押金支付
- 身份证上传
- 驾驶证上传
- 人脸识别
- 精确定位
- 真实订单合同
- 自动库存锁定
- 保险计算
- 违章处理

## 7. 隐私提示文案建议

预约页底部可放：

```text
提交预约即表示您同意我们仅将所填信息用于本次车辆预约沟通。车辆档期、价格、押金及取还车规则以客服最终确认为准。
```
