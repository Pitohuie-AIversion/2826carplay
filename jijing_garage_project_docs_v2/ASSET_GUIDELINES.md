# ASSET_GUIDELINES.md

## 1. 图片素材目标

车辆图片是本项目首页和详情页的核心视觉资产。

第一版可以使用本地占位图片，但真实上线前必须替换为统一风格车辆图片。

## 2. 图片目录

推荐目录：

```text
assets/
  cars/
    car_001_cover.jpg
    car_001_01.jpg
    car_001_02.jpg
    car_001_03.jpg

  icons/
    phone.png
    service.png
    share.png
    user.png
```

## 3. 命名规范

车辆图片命名：

```text
car_001_cover.jpg
car_001_01.jpg
car_001_02.jpg
car_001_03.jpg
```

规则：

- `cover` 用于首页车辆卡片
- 数字图片用于详情页轮播
- 文件名使用小写英文、数字和下划线
- 不使用中文文件名
- 不使用空格
- 不使用特殊符号

## 4. 图片规格

首页封面图建议：

```text
比例：16:9 或接近 2:1
宽度：750px 左右
格式：jpg / webp
大小：单张尽量小于 500KB
```

详情页图片建议：

```text
比例：16:9
宽度：1000px 左右
格式：jpg / webp
大小：单张尽量小于 800KB
```

## 5. 图片风格

推荐风格：

- 地下车库
- 暗色背景
- 车辆 3/4 视角
- 车身高光清楚
- 背景干净
- 色调统一
- 不要过曝
- 不要杂乱街景
- 不要包含无关人物

## 6. 使用规则

WXML 中车辆图片必须使用：

```text
mode="aspectFill"
```

禁止：

- 使用 base64 图片
- 使用超大 PNG
- 使用远程不稳定图片链接
- 图片被强行拉伸变形
- 首页一次性加载过多高清图片
