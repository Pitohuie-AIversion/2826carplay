# CONTENT_GUIDE.md

## 1. 文案风格

本项目文案风格应保持：

- 简洁
- 有车库氛围
- 有性能车感
- 不夸大
- 不承诺无法保证的服务
- 不使用过度营销话术

## 2. 品牌文案

推荐主标题：

```text
极境车库
```

推荐副标题：

```text
精选个性车型，预约你的下一次驾驶体验
```

## 3. 分类文案

推荐分类：

```text
迷你乐趣
奢享旗舰
性能闪电
经典燃油
电动先锋
```

分类含义：

```text
迷你乐趣：轻巧、有趣、适合城市和周边驾驶
奢享旗舰：舒适、豪华、商务或高端体验
性能闪电：动力强、操控感强、运动取向
经典燃油：机械感、燃油车、手动挡或经典车型
电动先锋：新能源、电动车、智能化车型
```

## 4. 状态文案

车辆状态：

```text
在库
出库
维护中
已预约
```

不要使用：

```text
马上抢
限时疯抢
全网最低
绝对安全
百分百可租
```

## 5. 预约成功文案

推荐：

```text
预约信息已提交，客服将尽快联系您
```

## 6. 空状态文案

当前分类无车辆时：

```text
当前分类暂无车辆，更多车型即将入库
```

车辆不存在时：

```text
未找到该车辆，请返回车库重新选择
```

## 7. 租赁说明文案

第一版可以使用：

```text
车辆价格、可用时间、押金和取还车规则以客服最终确认为准。
提交预约后，客服将与您确认车辆档期和具体租赁细节。
```

## 8. 禁止文案

禁止写入：

- 绝对最低价
- 包过审核
- 无需证件
- 无押金承诺
- 任何规避规则的表述
- 未经确认的保险承诺
- 夸大车辆性能的危险驾驶描述
