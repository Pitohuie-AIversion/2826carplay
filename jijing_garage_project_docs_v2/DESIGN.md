# DESIGN.md

## 1. 设计目标

本项目视觉目标是做成“高质感车库展示页”，而不是普通租车商城页面。

参考方向：

- 暗色车库背景
- 横向车辆大图
- 车辆昵称突出
- 状态标签清晰
- 价格信息直接
- 底部操作栏固定
- 整体风格偏性能车、车库、俱乐部感

## 2. 首页核心布局

首页结构从上到下：

```text
顶部标题：极境车库

分类导航：
迷你乐趣 | 奢享旗舰
性能闪电 | 经典燃油

车辆卡片列表：
车辆封面图
左上角车辆昵称
右上角库存状态
底部车型、标签、价格

底部操作栏：
客服 | 电话 | 分享 | 我的
```

## 3. 颜色规范

推荐主色：

```css
--bg-main: #101010;
--bg-panel: #181818;
--bg-card: #151515;
--border-card: #3a3a3a;

--text-main: #ffffff;
--text-sub: #b8b8b8;
--text-muted: #777777;

--blue-main: #2f80ed;
--blue-light: #56ccf2;
--gray-status: #8a8a8a;
--orange-status: #f2994a;
--purple-status: #9b51e0;
```

状态颜色：

```text
在库：蓝色
出库：灰色
维护中：橙色
已预约：紫色
```

## 4. 字体层级

```text
页面标题：34rpx - 40rpx，白色，中粗
分类文字：30rpx - 34rpx，选中高亮，未选中灰色
车辆昵称：42rpx - 52rpx，白色，粗体或斜体
车型名称：28rpx - 32rpx，白色
车辆标签：24rpx - 28rpx，白色或浅灰
价格：28rpx - 32rpx，白色，中粗
说明文字：24rpx - 28rpx，灰色
```

## 5. 车辆卡片设计

卡片比例：

```text
宽度：接近屏幕宽度，左右留 24rpx - 32rpx
高度：建议 360rpx - 420rpx
图片比例：接近 16:9
圆角：8rpx - 14rpx
边框：1rpx 深灰
```

卡片内部：

```text
左上角：车辆昵称
右上角：状态标签
底部：半透明黑色信息层
底部左侧：车型 + 标签
底部右侧：今日价格
```

图片要求：

```text
mode="aspectFill"
车辆主体清晰
横向构图
不要拉伸变形
不要过亮
背景尽量统一为车库、地下停车场、暗色场景
```

## 6. 交互设计

分类点击：

- 当前分类高亮
- 车辆列表即时筛选
- 不需要复杂动画

车辆卡片点击：

- 进入车辆详情页
- 路由携带 carId

电话按钮：

- 使用 `wx.makePhoneCall`

客服按钮：

- 优先使用微信客服能力

分享按钮：

- 使用小程序原生分享能力

## 7. 禁止设计方向

禁止做成：

- 白底商城
- 普通商品列表
- 新闻资讯页面
- 过度卡通化页面
- 强渐变背景页面
- 复杂玻璃拟态页面
- 元素太多的活动页
- 与汽车展示无关的营销页

## 8. 第一版设计验收

第一版完成后，首页应该满足：

- 一眼能看出是汽车车库展示
- 车辆图片是视觉主体
- 分类清楚
- 车辆状态清楚
- 价格清楚
- 底部入口清楚
- 整体接近用户提供的参考截图
