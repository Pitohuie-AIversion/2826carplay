# TODOlist.md

## 总规则

本 TODOlist 用于约束 Agent 分阶段开发微信小程序。

Agent 必须严格按照阶段顺序执行。  
每次只完成当前阶段任务。  
完成一个阶段后，必须停止并等待用户确认。  
不允许跳过阶段。  
不允许主动扩展功能。  
不允许一次性完成所有功能。  

---
